# Réponses : Expressions Arithmétiques en Java

Voici les explications et résultats pour chaque expression.

1. `x + y * z - gamma` → 14 + (6 * 3) - 2 = 30
2. `alpha / beta + gamma % z` → 10 / 4 + 2 % 3 = 2 + 2 = 4
3. `delta += epsilon * theta - omega` → 8 + (5 * 7 - 9) = 8 + 26 = 34
4. `omega /= x - y + z` → 9 / (14 - 6 + 3) = 9 / 11 = 0 (division entière)
5. `beta %= gamma + epsilon - theta` → 4 % (2 + 5 - 7) = 4 % 0 → Erreur (division par zéro)
6. `++alpha + beta * gamma - delta` → 11 + 4 * 2 - 34 = 11 + 8 - 34 = -15
7. `omega-- - epsilon + theta * z` → 0 - 5 + 7 * 3 = -5 + 21 = 16
8. `x *= y + z - gamma` → 14 * (6 + 3 - 2) = 14 * 7 = 98
9. `(alpha + beta) * (gamma - delta % epsilon)` → (11 + 4) * (2 - 34 % 5) = 15 * (2 - 4) = 15 * -2 = -30
10. `x++ + --y * z % alpha - beta / gamma` → 98 + 5 * 3 % 11 - 4 / 2 = 98 + 15 % 11 - 2 = 98 + 4 - 2 = 100
