Pour chaque opération, prédire l’affichage de la valeur choisie.
## Niveau 1 – Simple
### Exercice a)
```java

a += 3;
System.out.println("a = " + a);					a = __________
```

## Niveau 4 – Complexe
### Exercice  a)
```java
int i = 7;
int j = 4;
int k = 2;
int l = i++ + j * --k;
System.out.println("l = " + l); 					l = __________
System.out.println("i (après post-incrémentation) = " + i);	i = __________
```
